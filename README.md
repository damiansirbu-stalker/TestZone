# TestZone: Callback Monitoring and Profiling for STALKER Anomaly

Instruments all 162 engine callbacks with fire counts, payload logging, and deep introspection of table and userdata arguments. Rate limiting prevents log spam from high-frequency callbacks. Per-callback toggles and periodic statistics through MCM.

[Releases](https://github.com/damiansirbu-stalker/TestZone/releases)

Requires: Anomaly 1.5.3, Modded exes, [xlibs](https://www.moddb.com/mods/stalker-anomaly/addons/xlibs-1001), MCM
